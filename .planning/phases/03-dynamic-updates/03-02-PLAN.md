# Plan: 03-02 - Placements & NCC Achievement Showcase

## Goal
Create dedicated showcases for the college's biggest strengths: Career Placements and NCC Excellence.

## Requirements
- ACHIEVE-01: Placement partner logo wall.
- ACHIEVE-02: Student success count/statistics.
- NCC-01: Detailed wings information (Army/Navy).

## Proposed Changes

### Data Modeling
- Create `lib/data/placements.json`:
  ```json
  [
    { "name": "TCS", "logo": "...", "description": "Global IT leader" },
    ...
  ]
  ```
- Create `lib/data/achievements.json`:
  ```json
  [
    { "title": "RDC Parade Winner", "year": "2024", "wing": "Naval" },
    ...
  ]
  ```

### Components
- `components/PlacementWall.tsx`: Responsive marquee or grid of partner logos.
- `components/NCCWingCard.tsx`: Detailed card for Army/Navy wings.
- `components/SuccessStat.tsx`: Animated counter for placement numbers.

### Pages
- `app/achievements/page.tsx`: Combined page for Placements and NCC (or split into two if content is heavy).
- Update `app/page.tsx` to link to these new sections.

## Verification Plan

### Automated Tests
- Build check: `npm run build`

### Manual Verification
1. Verify the placement logos are legible and correctly sized.
2. Check the NCC section for correct wing descriptions.
3. Ensure links from the homepage to "Achievements" are functional.
