# Plan: 03-01 - Campus Photo Gallery

## Goal
Implement a visually stunning, filtered photo gallery to showcase campus life, infrastructure, and events.

## Requirements
- INFO-05: Responsive photo gallery component.
- CATEGORY-01: Filtering by Campus, Labs, NCC, Placements.
- UX-01: Lightbox interaction for full-screen viewing.

## Proposed Changes

### Data Modeling
- Create `lib/data/gallery.json`:
  ```json
  [
    { "id": "1", "url": "...", "caption": "Computer Lab", "category": "Labs" },
    ...
  ]
  ```

### Components
- `components/GalleryFilter.tsx`: Tab-based filtering UI.
- `components/GalleryGrid.tsx`: Staggered/Masonry-style grid for images.
- `components/Lightbox.tsx`: Client-side modal for image expansion.

### Pages
- `app/gallery/page.tsx`: The main gallery listing page.

## Verification Plan

### Automated Tests
- Build check: `npm run build`
- Type check: `npx tsc --noEmit`

### Manual Verification
1. Navigate to `/gallery`.
2. Verify all images load with placeholders.
3. Test filtering (click "NCC" and ensure only NCC images show).
4. Click an image and verify the Lightbox opens correctly.
5. Verify responsive layout on mobile (grid should collapse to 1-2 columns).
