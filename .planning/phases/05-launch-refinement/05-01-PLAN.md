# Plan: 05-01 - SEO & Accessibility Audit

## Goal
Optimize the site for search engines and ensure it meets basic accessibility standards.

## Requirements
- SEO-01: Metadata for all pages.
- ACC-01: Aria labels and alt tags.
- PERF-01: Final image optimization check.

## Proposed Changes

### Metadata
- Update `app/layout.tsx` with global SEO keywords.
- Ensure each page (`/about`, `/academics`, etc.) has a unique, descriptive title.

### Components
- Audit `Navbar.tsx`, `Footer.tsx`, and `Hero` for accessibility.
- Add `aria-label` to all icon-only buttons.

## Verification Plan

### Automated Tests
- Build check: `npm run build`

### Manual Verification
1. Inspect page head in browser for correct meta tags.
2. Verify all images have descriptive alt text.
3. Test site with a screen reader emulator if possible.
