<decisions>
## Implementation Decisions

### Technical Stack
- **Framework**: Next.js 14+ (App Router) — Standard for high performance and SEO.
- **Styling**: Tailwind CSS — Enables rapid, responsive UI development.
- **Language**: TypeScript — Ensures type safety for the portal features later.

### Brand & UI
- **Primary Colors**: Academic Navy Blue (#003366) and Gold (#D4AF37) / White.
- **Navigation**: Responsive Top Header with a mobile hamburger menu.
- **Typography**: Clean, professional sans-serif (e.g., Inter or Roboto).
