# Plan: 04-01 - Admissions & Inquiry System

## Goal
Implement a professional admissions information page and a functional inquiry form to capture potential student leads.

## Requirements
- ADMN-01: Admissions process documentation.
- ADMN-02: Multi-step inquiry form.
- UX-02: Form validation and success states.

## Proposed Changes

### Components
- `components/AdmissionsForm.tsx`: Interactive form with validation (Zod + React Hook Form).
- `components/AdmissionStep.tsx`: Vertical progress indicator for the admissions process.

### Pages
- `app/admissions/page.tsx`: Main admissions portal.

### Lib
- `lib/validations.ts`: Zod schemas for form validation.

## Verification Plan

### Automated Tests
- Build check: `npm run build`

### Manual Verification
1. Navigate to `/admissions`.
2. Fill out the form with invalid data (e.g., short phone number) and verify error messages.
3. Submit a valid form and verify the "Success" toast/message.
4. Check mobile responsiveness of the form fields.
