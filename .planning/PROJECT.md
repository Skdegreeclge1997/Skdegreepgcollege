# SK Degree College Website

## What This Is

A comprehensive website for SK Degree College that serves as both an informational hub and an action-oriented platform. It provides prospective students, current students, and faculty with access to course details, announcements, directories, and administrative tools like admission forms and student portals.

## Core Value

To seamlessly connect all stakeholders—students, parents, and faculty—with the college's resources, information, and updates in one accessible, reliable place.

## Context

- The platform must cater to a broad audience (everyone) ranging from tech-savvy current students to parents and administrative staff.
- Must be responsive and accessible across all devices (mobile is likely the primary way students will access the notice board).
- Project initialized in a previously mostly empty directory (`naveenb.html` is present).

## Constraints

- **Responsiveness**: Must work flawlessly on mobile and desktop — students heavily rely on mobile devices.
- **Maintainability**: The dynamic notice board and photo gallery must be easy for non-technical staff to update.
